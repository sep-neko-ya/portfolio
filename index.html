RESONARK X LITE (Oculus Quest)

Oculus Quest (無印/2) で動作するVRリズムゲームです。スマートフォンを用いることで、DMM VR Connect経由で自分のアバターを読み込むことが可能です。

Oculusプラットフォームでリリースするための最適化(無印Questで、基本的にフレーム落ちせずに動作し続けるという条件)が課題となったため、グラフィックスや処理の見直しを行いました。

利用技術

  • Unity 2019.4
    • Oculus Integration
    • DMM VR Connect SDK
    • Final IK
    • UniRx など
  • Android Studio

担当範囲

  • フロントエンド開発 (パフォーマンス調整, 他プラットフォームとの連携機能実装含む)
  • ストア向けパブリッシュ
  • 一部グラフィック制作
  • 広報 (Web制作、SNS運用)
  • その他、制作進行、翻訳、ユーザーサポート など

参考情報

  • Oculus Store / RESONARK X LITE
  • 電ファミニコゲーマー / 「拍手と糸」を使って演奏するOculus Quest／Quest 2向けVRリズムゲーム『レゾナーク クロス ライト』配信開始。好きなアバターで遊べるVRM連携機能を搭載

リリース日: 2021/4/22

RESONARK X (Steam)

Steamで動作するVR/Desktop向けリズムゲームです。VRモードでは、VRoid Hubやローカルから自分のアバターを読み込むことが可能です。

Desktop/VRのマルチで動作し、中長期的なアップデートに耐えられるシステム設計が課題となったため、データの持ち方や処理の流れに注意して設計/実装を行いました。

利用技術

  • Unity 2019.4
    • SteamVR Plugin
    • VRoid SDK
    • Final IK
    • Cinemachine
    • UniRx など
  • Twitter API

担当範囲

  • フロントエンド開発 (他プラットフォームとの連携機能実装含む)
  • ストア向けパブリッシュ
  • 一部グラフィック制作
  • 広報 (Web制作、SNS運用)
  • その他、制作進行、翻訳、ユーザーサポート など

参考情報

  • Steam / RESONARK X

RESONARK (VRChat World)

VRChat上のワールドとして動作するVR/Desktop向けリズムゲームです。VRモードでは、VRoid Hubやローカルから自分のアバターを読み込むことが可能です。

2021/9/15時点で延べ40万人以上のVisit数を記録しています(RESONARK 無印 ～ 3 の合算)。一時期、VRChat運営に「Featured World Udon」として取り上げていただいたこともありました。

利用技術

  • Unity 2019.4 / VRChat SDK3 (Udon)
    • UdonSharp
    • Cinemachine など

担当範囲

  • ワールド制作/運営 (他プラットフォームとの連携機能実装含む)
  • 一部グラフィック制作
  • 広報 (Web制作、SNS運用)
  • ワールドを利用したイベント運営
  • その他、制作進行、翻訳、ユーザーサポート など
